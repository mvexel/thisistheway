About "This Is The Way"

This little tool aims to give you one place to inspect any OSM node, way or relation. It was born out of my own frustration of having to look different places for bits of information I know to be out there but are spread out in different tools.

To use, enter a node, way or relation id in the input field and hit "GO". The accepted input formats are:

  • n, w or r followed by the id, for example r52360
  • node, way or relation followed by / and the OSM id, for example relation/52360

If the input is valid and the object exists, you will see a map with the object highlighted, information about when and by whom it was last modified and in which changeset, as well as information about its tags and edit history.

The URL contains the feature id, so you can share it with others. If you link to the tool with the feature id in the location hash (like https://way.osm.lol/relation/52360), the tool will pick this up and retrieve the information right away.

If you have any questions or feature requests, please write to me at m@rtijn.org. The code lives here.
